Kenai National Wildlife Refuge Species List

Summary

This repository exists for cataloging species known to be present on the Kenai National Wildlife Refuge.

It also includes a script for converting checklist data from FWSpecies to Darwin Core Archive checklist format.

The directory structure is being conformed to the recommended file structure of the Alaska Region Data Stewardship Team (2019).

References
